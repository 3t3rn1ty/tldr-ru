# echo

> Команда вывода.

- Вывести текстовое сообщение. Кавычки могут быть опущены:

`echo {{"Hello World"}}`

- Вывести сообщение, используя переменную окружения:

`echo {{"My path is $PATH"}}`

- Вывести сообщение без перевода строки в конце строки:

`echo -n {{"Hello World"}}`

- Включить интерпретацию обратных слэшей (специальные символы, в примере -- \t = табуляция):

`echo -e {{"Column 1\tColumn 2"}}`
