# git show

> Отобразить различные типы объектов git (коммиты, теги и т.д.).

- Показать информацию о последней фиксации(коммите) (сообщение, изменения и другие метаданные):

`git show`

- Показать информацию о данной фиксации(коммите):

`git show {{commit}}`

- Показать информацию о фиксации(коммите), связанной с данным тегом:

`git show {{tag}}`

- Показать информацию о 3-м коммите с конца ветки:

`git show {{branch}} ~ {{3}}`

- Показать хэш и сообщение фиксации(коммита) в одной строке, подавляя вывод различий:

`git show --oneline -s {{commit}}`
