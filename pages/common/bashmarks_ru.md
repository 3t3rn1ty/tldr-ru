# bashmarks

> Сохраняет и прыгает по часто используемым папкам, используя однобуквенные команды.

- Список закладок:

`l`

- Сохраняет текущую папку как "имя_закладки":

`s {{имя_закладки}}`

- Переходит по закладке:

`g {{имя_закладки}}`

- Печатает содержимое папки с закладкой:

`p {{имя_закладки}}`

- Удаляет закладку:

`d {{имя_закладки}}`
