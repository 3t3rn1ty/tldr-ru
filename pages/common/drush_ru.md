# drush

> Оболочка командной строки и интерпретатор скриптов для Drupal.

- Загружает модуль "foo":

`drush dl {{foo}}`

- Загружает модуль "foo" с версией 7.x-2.1-beta1:

`drush dl {{foo}}-7.x-2.1-beta1`

- Включает модуль "foo":

`drush en {{foo}}`

- Выключает модуль "foo":

`drush dis {{foo}}`

- Очищает весь кэш:

`drush cc all`

- Очищает кэш CSS и JavaScript:

`drush cc css-js`
