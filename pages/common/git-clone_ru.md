# git clone

> Клонировать существующий репозиторий.

- Клонировать существующий репозиторий:

`git clone {{remote_repository_location}}`

- Для клонирования с локального хоста:

`git clone -l`

- Клонировать в "тихом" режиме:

`git clone -q`

- Клонировать существующий репозиторий, отбросив ревизии, которые глубже 10-ти
коммитов. При работе с большими репозиториями может сэкономить много времени:

`git clone --depth 10 {{remote_repository_location}}`
